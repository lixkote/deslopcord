# deslopcord
discord but not bloated

### Full
Docks the floating user panel, brings back the old server icons, and cuts out unnecessary stuff (gifts, billing settings, nitro mentions here and there). Also strips out most ads, upsells, and other clutter.

You'll still be able to see nitro effects on other's profiles as before.\

**TL;DR: a clean, distraction-free discord experience.**\


### Lite
Removes the floating user panel and brings back the old server icons.
**Doesn't touch anything nitro related.**\
